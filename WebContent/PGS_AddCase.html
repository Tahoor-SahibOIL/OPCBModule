Welcome to the Project Gate System (PGS)

Use this form to create a new project gate case. The Project Gate System supports effective management of project approvals across defined gate stages. Please review the steps below before proceeding:

  • Gate I: Initiate and Define Idea
  • Gate II: Identification
  • Gate III: Feasibility Study
  • Gate IV: Front End Engineering Design (FEED)
  • Gate V: Mechanical Completion
  • Gate VI: Close-out

Note: Please ensure all required information is entered accurately before submitting the case for review.
